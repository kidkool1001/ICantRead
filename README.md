# I Can't Read

Tiny mod for SPT that changes the 24hr raid times to 12hr with AM/PM. 

---

### Example
- **Before:** `18:45:36`  
- **After:** `06:45:36 PM`

---

### Install
1. Drop `ICantRead.dll` into your `BepInEx/plugins` folder.  
2. Launch the game.  
3. Stop pretending you know what 23:00 means.
